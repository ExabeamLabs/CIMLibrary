alert-modify

-----------


Parameters
----------
| Parameter     | Value        |
| ------------- | ------------ |

Fields
------

The possible fields for this activity type will vary depending on whether the activity was a [success](#alert-modifysuccess) or a [fail](#alert-modifyfail).

--------------------
